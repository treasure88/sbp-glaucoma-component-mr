# Methods reproducibility notes

Core workflow stages:

1. Data preparation and GWAS standardization.
2. Harmonization and instrument preparation.
3. Primary two-sample MR.
4. IOP-dependent minus IOP-independent component contrast.
5. Robustness and pleiotropy-oriented sensitivity analyses.
6. External ophthalmic triangulation.
7. Minimum detectable-effect analysis.
8. Table and figure generation.

The repository should not be interpreted as containing raw data or as providing patient-level clinical guidance.
