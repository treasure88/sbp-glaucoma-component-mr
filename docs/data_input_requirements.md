# Data input requirements

This repository contains analysis code, metadata, and reproducibility documentation only.

Raw GWAS summary statistics and LD reference panels are not redistributed. Users must obtain source GWAS data from the original providers and place prepared inputs under `data/prepared_inputs/`.

The manuscript-level interpretation remains hypothesis-generating and non-confirmatory. The code is intended to reproduce component-contrast analyses, sensitivity analyses, external triangulation, and table/figure generation where source data access permits.
